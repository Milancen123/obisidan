# Spacing System

## Purpose
Defines all margin, padding, and spacing rules across the project using a consistent scale.

---

## 1. Spacing Scale

- space-4 → 4px
- space-8 → 8px
- space-12 → 12px
- space-16 → 16px
- space-24 → 24px
- space-32 → 32px
- space-40 → 40px
- space-48 → 48px
- space-64 → 64px
- space-80 → 80px
- space-96 → 96px

---

## 2. Global Padding

### padding-global
- padding-left: 120px
- padding-right: 120px

---

## 3. Section Padding

### padding-section
- padding-top: 100px
- padding-bottom: 100px


---

## 4. Element Spacing (Vertical Rhythm)

### gap-xsmall
- gap: 12px
### gap-small
- gap: 24px
### gap-medium
- gap: 32px
### gap-large
- gap: 48px
### gap-xlarge
- gap: 64px
### gap-xxlarge
- gap: 100px

---

## 5. Margin Utilities

### margin-top-small
- margin-top: 16px

### margin-top-medium
- margin-top: 24px

### margin-top-large
- margin-top: 40px

---

## 6. Rules

- Only use spacing from the scale
- No random values (18px, 22px, etc.)
- Prefer gap over margin when using flex/grid
- Keep vertical rhythm consistent across sections
